# My Enumerable Methods - Ruby

The my_enumerable methods used are only meant to replicate the ruby enumearble methods.


## my_enumerable methods include:

- my_each
- my_each_with_index
- my_all?
- my_none?
- my_any?
- my_select
- my_count
- my_map
- my_inject
